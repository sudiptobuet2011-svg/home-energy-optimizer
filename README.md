# Home Energy Optimizer

This project models the energy management of a single-family household with flexible assets such as PV, battery, EV, and heat pump.

Using day-ahead electricity prices and historical household consumption data, the model determines when each element should operate to minimize electricity cost over a 24-hour horizon.

The optimization considers technical constraints such as battery capacity, EV charging windows, and required heat pump energy.

## Goal

Demonstrate a simplified residential energy optimization model under dynamic electricity prices.
